# Telecom X - Análisis de Evasión de Clientes (Churn)

## 📋 Descripción
Proyecto de análisis de datos para identificar factores que influyen en la evasión de clientes de Telecom X y desarrollar modelos predictivos para la retención.

## 📊 Resultados Clave
- **Tasa de evasión actual:** 26.54%
- **Mejor modelo:** Regresión Logística (79.6% accuracy)
- **Factores de mayor riesgo:** Contrato mensual (42.7% evasión), Electronic check (45.3%), Fibra óptica (41.9%)
- **Factores de retención:** Contrato 2 años (2.8% evasión), Antigüedad > 37 meses

## 📁 Archivos
- `TelecomX_Analisis_Churn.ipynb` - Notebook completo con análisis, visualizaciones y modelos

## 🛠️ Tecnologías
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Scikit-learn (Regresión Logística, Random Forest)
- Google Colab

## 📌 Conclusiones Principales
Los clientes con mayor probabilidad de evasión son aquellos con:
- Contrato mensual
- Pago con Electronic check
- Internet de fibra óptica
- Menos de 18 meses de antigüedad
- Sin pareja ni dependientes
